Aula: Variáveis

📦 Variáveis em Java

Aprenda como armazenar diferentes tipos de dados
🔢 Números inteiros (Desbloqueado)

Usado para armazenar números sem casas decimais.

int idade = 18;
int ano = 2026;
int pontos = 150;
🔒 💰 Números decimais

Usado para valores com casas decimais.

double salario = 2500.50;
double altura = 1.75;
double preco = 19.99;
🔒 🔤 Texto (String)

Usado para armazenar palavras ou frases.

String nome = "Lucas";
String cidade = "Natal";
String curso = "Java";
🔒 ✅ Boolean (verdadeiro/falso)

Representa valores lógicos.

boolean ativo = true;
boolean logado = false;
boolean aprovado = true;
🔒 🧠 Exemplo completo
public class Main {
  public static void main(String[] args) {

    String nome = "Ana";
    int idade = 20;
    double altura = 1.68;
    boolean estudante = true;

    System.out.println(nome);
  }\n}
