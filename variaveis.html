📦 Variáveis em Java

Aprenda como armazenar informações no programa

O que são?

Variáveis são espaços na memória que guardam dados como números, textos e valores.

int idade = 18;
String nome = "Lucas";
boolean ativo = true;
Exemplo prático

Criando uma variável e exibindo no console:

public class Main {
  public static void main(String[] args) {
    String aluno = "Ana";
    System.out.println(aluno);
  }
}
Dica

Sempre use nomes claros para suas variáveis para facilitar o entendimento do código.
